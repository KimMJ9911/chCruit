# chCruit

방송 플랫폼 방송인들을 위한 구인/구직 매칭 서비스

![chCruit Logo](https://via.placeholder.com/200x80/10B981/FFFFFF?text=chCruit)

## 📋 프로젝트 개요

**chCruit**은 방송 플랫폼의 방송인과 크리에이터를 위한 전문 구인/구직 매칭 플랫폼입니다. 편집자, 채팅 매니저, 썸네일러 등 방송 제작에 필요한 다양한 직종의 인재를 연결합니다.

### 주요 타겟

- **구인자**: 편집자, 매니저, 디자이너 등이 필요한 방송인 및 크리에이터
- **구직자**: 방송 제작 관련 업무를 찾는 전문가 및 프리랜서

## ✨ 주요 기능

### 1. 회원 관리
- **일반 회원가입**
  - 이메일/비밀번호 기반 회원가입
  - 이메일 인증 시스템
  - 닉네임 중복 확인
  - 프로필 설정 (구인자/구직자 선택, 직종, 자기소개)
  
- **소셜 로그인**
  - 구글 로그인 (OAuth 연동)
  - 네이버 로그인 (OAuth 연동)
  - 이메일 인증 생략, 바로 프로필 설정

- **로그인/로그아웃**
  - 세션 관리
  - 보호된 라우트 (로그인 필요 페이지 자동 리다이렉트)

### 2. 구인/구직 게시판
- **구인 게시판**
  - 방송인이 필요한 인재 모집
  - 직종별 필터링 (편집자, 채팅 매니저, 썸네일러, 기타)
  - 상세 조건 입력 (급여, 근무 형태, 우대사항)
  
- **구직 게시판**
  - 크리에이터가 구직 정보 게시
  - 포트폴리오 및 이력서 파일 첨부 (PDF, PPT, URL)
  - 영상 소스 공유 (URL)

- **검색 및 필터**
  - 직종별, 키워드별 검색
  - 정렬 기능 (최신순, 조회순)

### 3. 자유 게시판
- **비로그인 글쓰기**
  - 로그인 없이도 게시글 작성 가능
  - 임시 닉네임 설정
  - 커뮤니티 활성화

### 4. 1대1 채팅
- **실시간 메시징**
  - 구인자와 구직자 간 직접 소통
  - 파일 공유 기능
  - 채팅 히스토리 관리

### 5. 매칭 이력
- **매칭 관리**
  - 지원 내역 조회
  - 매칭 상태 추적 (대기, 진행 중, 완료)
  - 과거 매칭 이력

### 6. 마이페이지
- **프로필 관리**
  - 개인정보 수정
  - 프로필 이미지 업로드
  - 자기소개 및 경력 관리
  
- **내 활동**
  - 작성한 게시글 관리
  - 채팅 목록
  - 매칭 이력

## 🛠 기술 스택

### Frontend
- **React** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 스타일링
- **Lucide React** - 아이콘
- **React Router** - 라우팅
- **Context API** - 상태 관리

```bash

npm run dev

# 빌드 미리보기
npm run preview
```
